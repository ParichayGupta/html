CSS size notes

  • em - This font size is calculated for the element relative to font-size of it's parent element (container).

  • For ex: If Font size of container is 10px and we give font-size to paragraph 3em then it will calculated as 10*3 that is 30px.

  • But if we apply padding or margin to the element then, the padding/margin will depend on the font-size of the same element not to the parent element.

  • For ex: If we give padding of 3em to the same element above having calculated font-size of 30px then the padding will be calculated as 30*3 that is 90px.

  • We can see the size computation in the computed section of chrome dev-tools.

  • rem - This font size is calculated for the element relative to font-size of html tag.

  • For ex: If Font size of html tag is 25px and we give font-size to paragraph 3em then it will calculated as 25*3 that is 75px.

  • Margin and padding will also calculated same as font-size.

  • For ex: if we give padding of 3rem to the same element above having 3rem font-size, the padding will be calculated as font-size of html tag* padding in rem that is 75px.

  • vh - used to give height for the element according to the screen size.

  • 100vh means full size of the display. 50vh means half of the screen size.

  • vw is works similar as vh, the only difference is it changes width instead of height.
