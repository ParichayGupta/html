Flex-box properties

  • Flex is a CSS display property which divides items (elements) into rows and columns.

  • Default flex-direction is row, but we can use column. row-reverse, column-reverse etc.

  • Flex wrap property will help to make the flex-items responsive. By default its nowrap.

  • justify-content property will help items to take equal space.

  • alilgn-items property will help align item to different locations. By-default it's flex-start but we can align items with different attributes like cneter, flex-end, flex-start.

  • align-items: stretch will automatically adds even spaces vertically between items to keep it responsive.
