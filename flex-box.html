Flex-box properties

  • Flex is a CSS display property which divides items (elements) into rows and columns.

  • Default flex-direction is row, but we can use column. row-reverse, column-reverse etc.

  • Flex wrap property will help to make the flex-items responsive. By default its nowrap.

  • justify-content property will help items to take equal space.

  • align-items property will help align item to different locations. By-default it's flex-start but we can align items with different attributes like center, flex-end, flex-start.

  • align-items: stretch will automatically adds even spaces vertically between items to keep it responsive.

Flex-items properties

  • Order: - The higher the order the later the element will be displayed

  • flex-grow: When we increase the screen width/size the element with higher flex-grow value will take more space in comparison with other elements.

  • flex-shrink: When we decrease the screen width/size the element with higher flex-grow value will take less space in comparison with other elements.

  • Negative numbers are not allowed in the flex-grow and flex-shrink properties.

  • This property is used to manipulate the width or height of the element.

  • When flex-direction is row, flex-basis will control the width of the element.

  • When flex-direction is column, flex-basis will control the height of the element.

  • shorthand- flex: (flex-grow flex-shrink flex-basis);

  • align-self: item will align itself to different postions like flex-start, flex-end, center etc.
